# Node.js Study

## 이벤트 루프

### 용어

- 메모리
- 호출 스택 (LIFO)
- 백그라운드 (먼저 실행되었어도 호출 스택에 있는 것이 먼저 실행)
- 태스크 큐

#### 실행 순서

1. 메모리에서 함수나 여러가지 실행
2. 호출 스택에 쌓이게 된다.
3. 비동기 로직이 있을 경우 백그라운드에 들어간다.
4. 백그라운드에 있는 것이 태스크 큐로 이동하게 된다.
5. 호출스택이 비어있을 경우 태스크 큐에 있는 것이 다시 호출스택으로 이동하여 실행된다.
